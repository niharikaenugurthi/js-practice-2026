# 🎨 Background Color Changer

A simple and interactive project where users can change the background color by clicking on color boxes.

---

## 🚀 Features
- Click on different color boxes to change background
- Smooth UI with hover effects
- Beginner-friendly project
- Built using pure HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
